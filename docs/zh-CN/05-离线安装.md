# 离线安装

如果 Docker 镜像、GitHub、Gitee、Hugging Face 都不稳定，推荐使用 OpenDeepSeek CN 离线包。

## 1. 应下载什么

按系统和 CPU 选择：

| 系统 | CPU | 项目包 | 镜像包 |
|---|---|---|---|
| macOS Apple Silicon | arm64 | `opendeepseek-cn-v0.5.0-macos-arm64.zip` | `opendeepseek-images-cn-v0.5.0-arm64.tar.zst` |
| macOS Intel | amd64 | `opendeepseek-cn-v0.5.0-macos-amd64.zip` | `opendeepseek-images-cn-v0.5.0-amd64.tar.zst` |
| Windows | amd64 | `opendeepseek-cn-v0.5.0-windows-amd64.zip` | `opendeepseek-images-cn-v0.5.0-amd64.tar.zst` |
| Linux | amd64 | `opendeepseek-cn-v0.5.0-linux-amd64.tar.gz` | `opendeepseek-images-cn-v0.5.0-amd64.tar.zst` |
| Linux | arm64 | `opendeepseek-cn-v0.5.0-linux-arm64.tar.gz` | `opendeepseek-images-cn-v0.5.0-arm64.tar.zst` |

## 2. macOS / Linux

解压项目包后进入目录：

```bash
cd opendeepseek-cn
```

加载镜像并启动：

```bash
OPDS_CN_OFFLINE=./images/opendeepseek-images-cn-amd64.tar.zst bash install-cn.sh
```

Apple Silicon 把文件名换成 arm64。

## 3. Windows

推荐使用 WSL2 + Docker Desktop。

在 WSL Ubuntu 里：

```bash
cd opendeepseek-cn
OPDS_CN_OFFLINE=./images/opendeepseek-images-cn-amd64.tar.zst bash install-cn.sh
```

PowerShell 入口：

```powershell
.\install-cn.ps1
```

Windows 版 PowerShell 暂不自动解压 `.tar.zst`，如果用离线镜像包，请先解压成 `.tar` 后执行：

```powershell
docker load -i .\opendeepseek-images-cn-amd64.tar
.\install-cn.ps1
```

## 4. API Key

首次安装会要求填写 DeepSeek API Key。

它只会写入本机 `.env`：

```text
DEEPSEEK_API_KEY=sk-...
```

`.env` 不应该提交到 Git，也不会被上传到 OpenDeepSeek 仓库。

## 5. 文件权限

CN 默认建议把 Agent 权限收窄到：

```text
~/OpenDeepSeek-Agent
```

里面会有：

```text
OpenDeepSeek-Inputs
OpenDeepSeek-Outputs
OpenDeepSeek-Memory
```

如果你希望 Agent 读取桌面、下载或文档，请在后续 Portal/onboarding 里明确授权，不要默认全盘开放。

## 6. 验证

启动后访问：

```text
http://localhost:3000
```

检查配置：

```bash
./setup.sh verify
python3 scripts/benchmark_routing.py
```

完整验证需要服务已启动：

```bash
bash scripts/smoke-test.sh
```

## 7. 卸载

停止服务：

```bash
docker compose -f docker-compose.cn.yml down
```

如果要删除容器数据：

```bash
docker compose -f docker-compose.cn.yml down -v
```

注意：`down -v` 会删除 OpenWebUI/Hermes 的 Docker volume。执行前请确认不需要历史聊天和 Agent 数据。
